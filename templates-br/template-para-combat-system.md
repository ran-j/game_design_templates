# Template para Combat System
   
```markdown
[NOME DO SISTEMA: Nome do Sistema de Combate]

1. **Tipo de Combate**:
   - [Combate corpo a corpo, à distância, defensivo?]
   - [Como o combate contribui para a tensão do jogo? O jogador pode lutar diretamente ou precisa evitar os inimigos?]

2. **Mecânica de Ataque**:
   - **Ataques Básicos**: [Quais tipos de ataques o jogador pode realizar?]
   - **Armas Disponíveis**: [Quais armas podem ser usadas? Cada uma tem efeitos específicos?]
   - **Dano Causado**: [Quanto dano o jogador pode infligir?]

3. **Mecânica de Defesa**:
   - **Esquivas e Bloqueios**: [O jogador pode se esquivar ou bloquear ataques?]
   - **Recursos Defensivos**: [O jogador pode usar itens ou habilidades para evitar o combate ou se defender?]

4. **Limitações e Dificuldades**:
   - [Quais são as limitações do jogador em combate? Baixa munição, armas frágeis, dificuldade em acertar os inimigos?]

5. **Impacto no Jogador**:
   - [O combate tem algum impacto na saúde física ou mental do jogador? Isso pode incluir efeitos como medo, desespero, ou perda de sanidade.]
```
