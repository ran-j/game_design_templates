# Template para Objetivos no Jogo

```markdown
[NOME DO OBJETIVO: Nome do Objetivo]

1. **Tipo de Objetivo**:
   - [Sobrevivência, exploração, resolução de quebra-cabeça, fuga, combate, encontrar itens, etc.]

2. **Descrição**:
   - [Descrição geral do que o jogador precisa fazer. Isso pode envolver descobrir algo, chegar a um local específico, derrotar uma entidade, etc.]

3. **Contexto na História**:
   - [Explique como esse objetivo (caso aplicável) se relaciona com a narrativa geral. O jogador está tentando escapar de algo? Desvendar um segredo? Reverter uma maldição?]

4. **Desafios e Ameaças**:
   - [Quais obstáculos ou perigos o jogador enfrentará enquanto tenta completar este objetivo? Isso pode incluir inimigos, quebra-cabeças, escassez de recursos, etc.]

5. **Recompensa**:
   - [O que o jogador ganha ao completar o objetivo? Pode ser progresso na história, um item essencial, uma nova arma ou a revelação de um segredo importante.]

6. **Interação com o Ambiente**:
   - [Como o jogador interage com o cenário ou os inimigos enquanto tenta alcançar o objetivo? Pode envolver ativar mecanismos, se esconder, lutar, explorar áreas perigosas.]

```
